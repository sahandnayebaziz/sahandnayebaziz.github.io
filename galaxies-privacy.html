Storing Data

Galaxies only stores the data it needs to function properly. You provide an email address, a username, and a password when signing up. These three values are saved and these only. Galaxies does not save your IP address or your location.

This is important: When you add an image to your personal Galaxy, the image is stored securely with a random filename in a file store that is not publicly-accessible for exploration. When you change images in your personal Galaxy, the original images are not deleted as they could be images that another Galaxies user has saved in the "Saved Galaxies" section of Galaxies.

Saving is an important part of Galaxies because it allows for one user to save another's Galaxy as-is on a certain date and keep that Galaxy viewable for as long as they would like. Only a Galaxy's current images (up to 4) are viewable at any given time by navigating to a user's Galaxy, so the saving feature lets users remember and capture photos shared by other users when they like something that was posted while still allowing for users to change their own Galaxy without leaving a long trail of images always showing.

Also important: Images shared to Galaxies by one user can be saved to a device by another user by a save button or by taken a screenshot of an image. These images are out of Galaxies' control.
